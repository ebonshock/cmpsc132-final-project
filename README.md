# Tic-Tac-Toe — CMPSC 132 Final Project
# Author: Evan Bonshock

A terminal-based 2-player Tic-Tac-Toe game written in Python.

## Features
- Colorized board display using ANSI escape codes
- Custom player names
- Persistent scoreboard across rounds
- Undo last two moves (type `u` at the row prompt)
- Replay last round move-by-move after each game

## How to Run
python tictactoe.py

## Controls
- On run enter `1` for a standard two player game or enter `2` to quit
- Enter row (0-2) then column (0-2) to place your mark
- Type `u` at the row prompt to undo the last two moves
- Enter `y` or `n` when prompted to show replay of round
- Enter `y` or `n` when prompted to play again
